Graph Visualization of Wikipedia

This website is to experiment and display graphs as a part of the development of the project of visualization of a sub-set of Wikipedia pages. This project is linked to a university project at EPFL

The dataset

Each node represents a Wikipedia page, each link or edge represents the hyperlinks to which one can be directed to from another page.
